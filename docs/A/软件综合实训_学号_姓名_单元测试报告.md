- 测试对象：`scripts/prepare_task_a_seed_data.py`
- 测试目的：验证种子数据生成脚本语法是否正确。
- 测试方式：使用 Python 编译检查。

```bash

```

- 预期结果：脚本无语法错误，命令正常结束。

### 3.2 测试项 2：Python 仿真脚本语法检查

- 测试对象：`scripts/run_task_a_simulation.py`
- 测试目的：验证离线仿真脚本语法是否正确。

```bash
python3 -m py_compile scripts/run_task_a_simulation.py
```

- 预期结果：脚本无语法错误，命令正常结束。

### 3.3 测试项 3：Java 后端编译测试

- 测试对象：`backend/`
- 测试目的：验证 Spring Boot 后端工程是否可以成功编译。
- 测试方式：使用 Maven 编译。

```bash

```

- 预期结果：后端代码编译通过，不出现 Java 语法错误或类型错误。

### 3.4 测试项 4：离线仿真脚本运行测试

- 测试对象：`scripts/run_task_a_simulation.py`
- 测试目的：验证成员 A 的核心仿真逻辑是否可以产生完整结果。

  2. 输出中包含 `runId`、`status`、`profile`、`scenario`、`timePoints`、`metrics`

### 3.5 测试项 5：结果结构检查

- 测试目的：验证仿真输出结构是否满足接口规范。
